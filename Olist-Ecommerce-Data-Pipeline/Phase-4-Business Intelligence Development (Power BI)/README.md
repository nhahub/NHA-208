# Olist Power BI Dashboard

## Overview
This folder contains the interactive **Power BI dashboard** for the **Olist E-Commerce Data Pipeline** project.  
The dashboard provides insights on sales, customers, products, and returns, and is connected to the MySQL database for real-time analysis.

## Notes
- The file should be opened using **Power BI Desktop**.  
- Once new data is loaded into the MySQL database, performing a **Refresh** in Power BI will update all tables and measures automatically.
